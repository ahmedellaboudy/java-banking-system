# Java Banking System

A simple console-based banking application built in Java.
It allows users to manage a basic bank account directly from the terminal.

## Current Features

- Check account balance
- Deposit funds
- Withdraw funds
- View transaction history

## How to Run

1. Clone the repository
2. Open the project in your preferred Java IDE (IntelliJ, Eclipse, VS Code)
3. Run `Main.java`

## ⚠️ Under Development

This project is still in its early stages. Planned features for future versions include:

- **Database integration** to persist account data between sessions
- **Graphical User Interface (GUI)** for a better user experience
- **Multiple accounts** support
- **User authentication** (login & password)

## Built With

- Java
- Scanner & ArrayList (standard Java library)
